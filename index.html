Portfolio Project: Adaptive Learning System – Multi-Language Support

Project: Adaptive Learning System
My Contribution: I worked on implementing dynamic language support in the Adaptive Learning System, enabling the platform to deliver educational content and feedback in multiple languages. My tasks included modifying FSM logic, refactoring agent message handling, ensuring translation consistency across agents, and adding fallback mechanisms. The supported languages now include Telugu, Hindi, and Spanish.

Project Overview

The Adaptive Learning System uses multiple agents to deliver personalized education. My focus was on Issue #382: Expanding Language Support . This involved enabling each agent—including the Tutor, Problem Generator, and Verifier—to deliver content in the student’s preferred language.

What I Did

  • Refactored fsm_controller.py to dynamically load language configurations per session.
  • Extended the message templates to include Hindi and Spanish alongside Telugu.
  • Created fallback logic: if a translation is missing, English is used by default without breaking the flow.
  • Wrote automated tests to validate message formatting and accuracy across languages.
  • Collaborated with the UI team to ensure language switching was reflected in real-time in the chat interface.
