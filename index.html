Project Summary & My Contributions

This platform is designed to help students learn Indian history using a team of intelligent agents. The system includes six key agents that work together to deliver lessons, quizzes, feedback, and personalized learning paths. My primary contributions were:

  • Developed the ProblemGenerator Agent to create dynamic quiz content.
  • Implemented parts of the AnswerVerifier Agent for response evaluation and feedback.
  • Contributed to multi-agent communication workflow design and debugging.
  • Helped with testing and curriculum content validation.
